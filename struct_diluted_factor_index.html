Detailed Description

Struct that holds all information on which VdaggerV must be diluted with which random vector.

For rVdaggerV and rVdaggerVr the VdaggerV-operators are additionaly multiplied with random vectors. For both, VdaggerV and random index combinations there are lookuptables in OperatorLookup. This struct contains the id's of vdaggerv and ric which belong together.

Indices needed to uniquely identify Q1 objects

Because vdaggerv is diagonal in dirac space, gamma may be factored out and it proves useful to calculate and reuse

Q1 = rvdaggerv * gamma * peram

Indices needed to uniquely identify Q2 objects

Because vdaggerv is diagonal in dirac space, gamma may be factored out.

If the \( \gamma_5 \)-trick is used it proves useful to calculate and reuse

Q2 = \( \gamma_5 \) peram1 \( ^\dagger \gamma_5 \) * vdaggerv * gamma * peram2

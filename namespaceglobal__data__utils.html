Makes an operator list object from a string

Parameters
operator_stringOperator as specified in the infile: A ';'-sperated list with individual operators. The individual operators are composed of '.'-seperated parts. E.g.
operator_list = g4.d0.p(0,0,1);g5.d0.p0,1
Momenta (p) can be specified as 3-momentum or by (one or more) scalar number(s). In the latter case all 3-momenta with corresponding absolute value are constructed
Returns
An Operator_list object constructed with the data obtained from operator_string

Internally uses boost to split the string and process the parts.

This is a factory function which returns by value and uses named return value optimization in order to call the constructor within the scope

Parameters
quark_stringQuark as specified in the infile quark = flavor : nb_rnd_vec : T diltT_type : dilT : E dilE_type : dilE : D dilD_type : dilD : path where the following abbreviations where used
  • flavor {u,s,c,b} : Quark flavor

  • dilT_type {I,B} : Dilution type in time

  • dilE : Number of dilution blocks in eigenvector space
  • dilD_type {I,B} : Dilution type in dirac space
  • dilD : Number of dilution blocks in Dirac space
